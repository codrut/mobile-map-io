# mobile-map-io
A boilerplate mobile hybrid app for I/O with maps

### Installation
1. Run ```npm install```
1. Run ```bower install```
1. Rename secrets.js.example to just secrets.js and add your server config
1. Import api/reports.sql
1. Run ```node api/server.js```
